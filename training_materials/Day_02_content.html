DAQmx concepts

  • Channel: ซอฟต์แวร์ที่เป็นตัวแทนของ physical channel และข้อมูลที่เกี่ยวข้อง (range, terminal config, scaling)
    • Analog Input channels (ai*)
    • Analog Output channels (ao*)
    • Counter I/O channels (ctr*)
    • Digital I/O channels (port*/line*)
    • Power channels
  • Task: รายการของ channels และข้อมูลที่เกี่ยวข้อง (timing, trigger)

Channel properties

  • Range: ย่านของค่าที่วัด/สร้าง เช่น ±10 V
  • Terminal config: วิธีต่อ เช่น singled-end
  • Scaling: สมการแปลงค่า

Task properties

  • Timing: คาบเวลา/ความถี่ในการทำงาน
  • Trigger: เงื่อนไขในการทำงาน

SCPI protocol

Standard Commands for Programmable Instruments is a standard for controlling programmable test and measurement devices


  • ASCII strings sent over physical layer (GPIB, RS232C, USB, Ethernet)
    • Commands as tree structure separated with :
    • Support Abbreviating (capital chars) and Concatenating (;)
    • Query commands end with ?
  • Example: MEASure:VOLTage:DC?;:MEAS:CURR:AC?

SCPI cmd ⇆ NI-VISA ⇆ instrument


  • อ้างอิงเครื่องมือวัดด้วย VISA reference
    • ตรวจสอบ interface/ชื่อ จาก NI MAX
  • ลำดับการโปรแกรมเครื่องมือวัด
    1. ใช้ VISA Open เปิดการเชื่อมต่อ
    2. วนรอบตั้งค่าด้วย VISA Write และอ่านค่าด้วย VISA Read
      • ตั้งค่า string เป็น ’' Codes Display
      • ใช้ Format Into String และ Fract/Exp String to Number เพื่อแปลง
    3. ใช้ VISA Close ปิดการเชื่อมต่อ

Programming UDP3305S


ลำดับของคำสั่งสร้างแรงดันไฟเลี้ยง

  1. :SOURce:Mode {NORMal|SER|PARA}
    • :SOURce:Mode NORMal ตั้งโหมดสร้างแรงดันปกติ
  2. [:SOURce#]:VOLTage[:LEVel][:IMMediate][:AMPLitude]
    • :SOURce1:VOLTage 2.0 ตั้ง CH1 ให้สร้างแรงดัน 2.0 โวลต์
  3. :OUTPut[:STATe] [CH1|CH2|CH3|SER|PARA|ALL,] {0|1|OFF|ON}
    • :OUTPut:STATe CH1, ON เปิด CH1 จ่ายเอาต์พุต

PicoScope 2206B


Specifications

  • 50 MHz bandwidth, 8-bit ADC, 32 MB capture memory
  • AWG 1 MHz bandwidth
  • Input range: ±20 mV, ±50 mV,…, ±10 V, ±20 V
  • Timebase: 2 ns/div, …, 5000 s/div

  • Triggers: edge, pulse width, rise/fall time, interval, …

Programming PicoScope 2206B

Sampling mode

  • Block mode: สุ่มวัดด้วย MAX sampling rate/{1,2,4,…} บันทึกในหน่วยความจำจนเต็ม แล้วรอโอนย้ายไปคอมพิวเตอร์
  • Streaming mode: สุ่มวัดแล้วโอนย้ายไปคอมพิวเตอร์
    • Compatible streaming: sampling rate ต่ำ, ไม่รองรับ trigger
    • Fast streaming: sampling rate สูง, รองรับ trigger
  • ETS (Equivalent Time Sampling) mode: หลังจาก trigger สุ่มวัดสัญญาณหลายรอบ บันทึกในหน่วยความจำจนเต็ม แล้วรอโอนย้ายไปคอมพิวเตอร์

ลำดับ PicoSDK: block mode

  1. PicoScope2000aOpen.vi เปิดใช้งาน
  2. PicoScope2000aSetDataBuffers.vi กำหนดหน่วยความจำข้อมูล
  3. PicoScope2000aSettings.vi กำหนด channel, timebase, trigger
  4. เข้าวงรอบ
    1. PicoScope2000aRunBlock.vi เริ่มสุ่มวัด
    2. PicoScope2000aIsReady.vi เช็คสถานะ
    3. PicoScope2000aGetValues.vi นำเข้าข้อมูล
  5. PicoScope2000aStop.vi หยุดการสุ่มวัด
  6. PicoScope2000aClose.vi ปิดการเชื่อมต่อ
